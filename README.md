Bienvenido al gestor de asistencia 1.1! hecho con php, javascript y html. Aquí podrás realizar el crud completo de los roles y los miembros que irán en el registro, junto con un apartado para registrar la asistencia de un miembro al igual que su hora de salida.

ACLARACIONES:
- No puedes ingresar números de teléfonos repetidos en miembros, como tampoco podrás ingresar nombres repetidos en roles.
- Solo podrás realizar 2 registros al día por miembro, si hay uno activo no podrás realizar otro con el mismo miembro.

PARA MONTAR EL PROYECTO:
- Emplea el archivo adjunto "ad_asistencia.sql" para base de datos.
- Verifica en la clase conexion.php que los datos estén acorde, esta aplicación fue desarrollada con XAMPP, por lo que si usas MAMP, deberás ponerle contraseña (está por defecto sin contraseña, usando root).

Fuera de todo, muchas gracias por probar!
